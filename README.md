# chess-master

Another chess game in node.js


## Board

### Size

Board: H x 8

### Colors

a1 = black
h1 = white
a8 = white
h8 = black

## Pieces

### Colors

bottom : white
up : black

### Types

* 1 king
* 1 queen
* 2 rook, alias la tour
* 2 knights, alias le cavalier
* 2 bishops, alias le fou
* 8 pawns
